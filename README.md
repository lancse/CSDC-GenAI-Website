# CSDC GenAI InfoHub
An informative and user-friendly website that explains: Frequently Asked Questions (FAQs) About Generative AI Ethical Use of Artificial Intelligence.
#
**Overview**

GenAI InfoHub is a simple website introducing to the basics of Generative AI.
It includes three sections/pages:

- Home: Intro to GenAI

- FAQs: Common questions and simple explanations

- Ethical Use: Guidelines and best practices for responsible AI use

- Articles: External articles for further learning

*This website features animated hero sections, floating visuals, interactive accordions, and responsive layouts.*
#
**How to Run the Project**
1. Download or clone the project folder.
2. Open the folder.
3. Locate the following files:

    index.html (Home)
    
    faq.html
        
    ethics.html
    
    articles.html
    
    plus the JS, CSS, and images inside /assets/


6. Open index.html in any web browser (Chrome recommended).
7. Navigate using the navbar and scrolling

#
**Frameworks / Libraries Used**

HTML5 – Main structure

CSS3 – Styling and animations

JavaScript – Interactivity

Bootstrap 5 – Navbar and responsive behavior

Google Fonts – Typography

Font Awesome – Icons
